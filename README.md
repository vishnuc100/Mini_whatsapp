## ▶️ Installation

```bash
git clone https://github.com/vishnuc100/Mini_whatsapp.git
cd Mini_whatsapp
npm install
```

Create a `.env` file:

```env
MONGO_URL=your_mongodb_connection_string
```

Run the project:

```bash
node app.js
```

Open in browser:

```
http://localhost:8080/chats
```
